LEXXI 0.2.0


lexxi provides an abstraction layer on top of what would otherwise be a normal express web app.

most of the tedious technical decisions are made for you (they're sensible, of course) so you can focus on the thing* you really want to create.

documentation will be available soon, i promise. read the quick start guide on github: https://github.com/hxgf/lexxi/



* by "thing," i mean something like a web application, rest api, competent web site, social community, store, forum, any combination thereof...whatever, really.
